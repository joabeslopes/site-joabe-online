Reaper scripts in Lua

  Scripts in Lua language, for automate the creation of multitrack show projects on the Reaper DAW.
  This is a very repetitive and time consuming work, so these scripts can fasten this work, by creating and coloring multiple tracks at once, reading all the music folders inside the project, and insert the audio files on the corresponding track, based on the file name.
  I decided to make then public and available for use in Reapack, an open source Reaper extension that allow the users to download scripts, effects, personalized themes and more, all made by the community.
  With Reapack installed, just import the link to my repository index.xml into it, then browse for "Joabe Lopes" and all my scripts will be there, with the instructions for use inside then.
