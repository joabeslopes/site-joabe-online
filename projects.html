Reaper scripts in Lua

  Scripts in Lua language for automating the creation of multitrack show projects on the DAW Reaper.
  This is a very repetitive and time consuming work, so this scripts can advance a good part of the service, by creating and coloring multiple tracks at once, reading all the folders inside the project main folder, and insert the audio files on the correct track, based on the file name.
  I decided not only to make then public, but also available for use in Reapack, an open source Reaper extension that allow the users to download scripts, effects, personalized themes and more, all made by the community.
  With the extension installed, just import my repository index.xml into the extension, then browse for "Joabe Lopes" and all my scripts will be there, with the instructions for use inside then.

Email search app in Java

  I developed an desktop app in Java, where the students search for the first name of the teacher, and the app shows his email and other relevant informations, like the id, surname, classes he teaches and it's periods.
  Furthermore, the app also have a button that copies the selected email, and open a new tab on the browser with the Outlook page, so that the students can send an email to the selected teacher.
  The app communicates directly with a MySQL database, through the JDBC driver, and send the SQL queries to get the necessary information.
  It's a simple app, but it was a fun experience that teaches me a lot of things.
